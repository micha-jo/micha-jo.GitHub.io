Qu'est ce qu'arduino?

Arduino® est un ensemble matériel et logiciel qui permet d'apprendre l'électronique (en s'amusant) tout en se familiarisant avec la programmation informatique. Arduino est en source libre; vous pouvez donc télécharger le schéma d'origine et l'utiliser pour élaborer votre propre carte et la vendre sans payer des droits d'auteur.

Le materiel

Une carte Arduino est un cerveau qui permet de rendre intelligent des systèmes électroniques et d'animer des dispositifs mécaniques.

Les Arduino® sont des cartes électronique programable ( dotées d'un processeur et de mémoire) qui vont ressevoir des données émises par un capteur (temperature, humidité, potentiomètres...) et va transmettre des données "un message" à des recepteurs (leds, moteurs...). vous pouvez les trouver sous plusieur tailles et sous plusieurs formes, les principaux type de carte sont les arduino Nano, arduino Uno et les arduino Mega.

Le logiciel

L'IDE Arduino est le logiciel qui permet de programmer les cartes Arduino. Les créateurs de Arduino ont développé un logiciel pour que la programmation des cartes arduino soit visuelle, simple et complète à la fois. C'est ce que l'on appelle une IDE, qui signifie Integrated Development Environment.

Le language de programation avec arduino

Un langage de programmation est un langage permettant à un être humain d'écrire un ensemble d'instructions (code source) qui seront directement converties en langage machine grâce à un compilateur (c'est la compilation). L'exécution d'un programme Arduino s'effectue de manière séquentielle, c'est-à-dire que les instructions sont exécutées les unes à la suite des autres.

Les fonctions

On connait déjà deux fonction: loop() et setup (). Une fonction est une séquence d’instructions réalisant un calcul ou une tâche. Une fonction peut posséder des paramètres d’entrée (des arguments) et peut également retourner des valeurs de sortie.

float moyenne (float a, float b){ //float = type de données, moyenne = nom de la fonction, a et b = deux varible différentes de type float.
  //instruction a réaliser lors de l'execution de la fonction             
  return (a+b)/2;                 //la fonction donne en sortie la moyenne de a et b.
}
void setup() {
  Serial.begin(9600); 
}
void loop() { 
  Serial.print(moyenne (4,8));    //on affiche le resultat de la fonction moyenne soit la moyenne de 4 et 8 donc 6. 
}

Les structures de contrôle

Les structures de contrôle sont des instruction dans le language de programations. Elles sont utilisées pour commander le déroulement d'un programme el le devier dans une certaine direction

L'instruction Si (if)

Le terme Si fait appelle à une hypothèse ou à un cas de figure précis qui déterminerait une décision d'action. if exécutera son bloc d'instructions seulement si la condition définit est remplie.

if (condition 1; condition 2...){       //Si la conditon 1 et la conditon 2 sont validé 
    Instruction 1;                        //Alors le programme réalise l'instruction 1 et l'instruction 2 
    Instruction 2;
    ...
    }
  else if (condition 3; condition 4...){ //Sinon si la conditon 3 et la conditon 4 sont validé 
    Instruction 3;                       //Alors le programme réalise l'instruction 3 et l'instruction 4
    Instruction 4;
    ...
    }
  else{                                  //Sinon
    Instruction 5;                       //Le progrmme réalise l'instruction 5. 
    ...
    }

L'instruction case (switch)

L'instruction switch / case permet au programmeur de construire une liste de "cas". Le programme teste chaque cas avec la variable de test, et exécute le code si une correpondance (un test vrai) est trouvé.

int expression = valeur; 
switch (expression) {
  case valeur 1:
    // instructions à exécuter si expression est égale à valeur 1
    break;
  case valeur 2:
    // instructions à exécuter si expression est égale à valeur 2
    break;
  case valeur 3:
    // instructions à exécuter si expression est égale à valeur 3
    break;
  default:
    // instructions à exécuter si expression n'est égale à aucune des valeurs des case
    break;
}

La boucle pour (for)

Cette fonction permet de répéter une action, un bout de programme autant de fois qu’on le souhaite.

for(/*initialisation de la variable*/ ; /*condition de la boucle*/  ; /*instruction à exécuter*/){
  // ligne de code à exécuter
  }

La boucle tant que (while)

En anglais, le mot « while » signifie « tant que ». Donc le programme que l’on va écrire, s’exécutera « tant que » la condition sera vraie (ou fausse).

while(condition1 ; condition2){
  Instruction...  // tant que les conditions sont respectées les instructions sont éxécuté
}

Les boutons

Un bouton poussoir est un interrupteur (ou contacteur) monostable : il retourne seul dans la position repos (« relâché »). Un bouton poussoir fournit une information logique (Appuyé / Relâché).

Les connecteur 1&3 et 2&4 sont relié entre eux.

Composants

#define bouton 2                                  //declaration de la variable boouton connecté à la broche 2

{

  Serial.println ("le bouton est appuyer");      //on affiche: "le bouton est appuyer" dans le port serie
}
}

Les joysticks

Le joystick est un capteur de position qui renvoie deux valeurs analogiques représentant sa position X,Y. Il peut servir d’interface pour naviguer dans un menu ou bien pour piloter un objet en direction ou en vitesse.

Le joystick est composé de deux potentiomètres couplés mécaniquement placés de manière à détecter les composantes horizontale et verticale du joystick.

Montage

Composants
  • Une carte arduino
  • Une platine d'essai (breadboard)
  • Des fils
  • Un joystick

Code

#define bouton 2                         // declaration de bouton connecter à la broche 2 
#define verticale A0                     // declaration du potentiometre verticale sur la broche A0
#define horizontale A1                   // declaration du potentiometre horizontale sur la broche A1

void setup() {
  pinMode(bouton, INPUT);                 //initialisation du bouton comme une entré
  digitalWrite(bouton, HIGH);             //on dit au bouton qu'il est en position élevé
  Serial.begin(9600);                     //initialisation du port serie 
}

void loop() {
  Serial.print("bouton:  ");              //on ecrit dans le port serie le texte entre guillmets 
  Serial.println(digitalRead(bouton));    //on lit la valeur numérique du bouton puis on l'ecrit dans le port serie 
  Serial.print("verticale: ");            //on ecrit dans le port serie le texte entre guillmets 
  Serial.println(analogRead(verticale);   //on lit la valeur analogique de la composante verticales et on l'affiche 
  Serial.print("horizontale: ");          //on ecrit dans le port serie le texte entre guillmets 
  Serial.println(analogRead(horizontale));//on lit la composotant horizontale puis on l'afiche 
  Serial.print("\n");                     //on saute une ligne dans le port serie 
  delay(500);                             //on attend 0.5 sec avant de reffaire le programme 
}

Les leds

Une LED est un composant capable d’émettre de la lumière lorsqu’il est parcouru par un courant électrique.

Elle ne laisse passer le courant électrique que dans un seul sens, pour que le courant puisse passer:

  • la cathode ("-" , patte courte) doit être connecté du coté de la masse (GND).
  • l’anode ("+", patte longue) doit être connecté du coté du port numérique.

Montage

Composants
  • Une carte Arduino
  • Une platine d'essai (breadboard)
  • Des fils
  • Une led rouge
  • Une resistance de 220 Ω

Code 1: Led qui clignotte

#define ledRouge 2            //Declaration de la led rouge sur la broche (pin) 2

void setup() {
  pinMode(ledRouge, OUTPUT);   //initialisation de ledRouge comme une sortie (output)
}

void loop() {
  digitalWrite(ledRouge, HIGH);//Allume la led, definie ledRouge sur le niveau haut
  delay(1000);                 //Le programme attend 1 seconde avant d'executer la prochaine instruction
  digitalWrite(ledRouge, LOW); //Etteind la led, definie ledRouge sur le niveau bas
  delay(1000);                 //Le programme attend 1 seconde avant d'executer la prochaine instruction
}

Les servo moteurs

Les servomoteurs sont des moteurs d'un type particulier, conçu pour faire tourner quelque chose jusqu'à une position bien précise et capable de maintenir cette position jusqu'à l'arrivée d'une nouvelle instruction.

En ajoutant deux resistance il est possible de le tranformer en un moteur capable de fonctionner en conitnue dans les deux sens.

Les capteur ultrasons

Le capteur à ultrasons (également souvent appelé sonar ou télémètre à ultrasons) détermine la distance d’un objet de la même manière que les chauves-souris ou les dauphins.

Le capteur permet une mesure d’un objet situé à une distance allant de 2cm à 4m et fourni une mesure avec une précision de 3mm. Principe de fonctionnement: (0)Un signal de déclenchement (trigger) est émis pendant 10µs (1)Le module émet 8 signaux (minimum) à 40kHz. (2)Le module se met à l’écoute d’un signal de retour (3)Le temps entre l’émission et la réception est le temps nécessaire au signal pour faire l’aller et le retour vers l’objet qui a réfléchi celui-ci

Montage

Composants
  • Une carte Arduino
  • Une platine d'essai (breadboard)
  • Des fils
  • Un capteur ultrasons

Code

#include "SR04.h" //on inclu la bibliotheque pour le capteur ultra sons

#define ECHO_PIN 2
SR04 sr04 = SR04(ECHO_PIN,TRIG_PIN);
long a;

void setup() {
   Serial.begin(9600);
}

   a=sr04.Distance();
   if (a<5){
     Serial.println("Vous êtes proche"); 
   }
  else {
    Serial.println("Vous êtes loin");
  }
  Serial.print(a);
  Serial.println("cm");
  delay(1000);
}

ressource bluetooth electronics

SoftwareSerial BTSerial(10, 11); // RX | TX  = > BT-TX=10 BT-RX=11

void setup()
{
  Serial.begin(9600);
  BTSerial.begin(9600);
}

void loop() {
  
  while(BTSerial.available()){
    message = BTSerial.readString();
    
    if(message == "A"){
      //instruction si l'application à envoyer "A" lors de l'appui sur un bouton /!\ au saut de ligne au espace 
      BTSerial.println("Le bouton est appuyé");
      Serial.println("Le bouton est appuyé");
    }
      if(message == "a"){
      //instruction si l'application à envoyer "A" lors de l'appui sur un bouton /!\ au saut de ligne au espace 
      BTSerial.println("Le bouton n'est pas appuyé");
      Serial.println("Le bouton est appuyé");
    }
  
      if(message.endsWith("d*\n")== true ){
    
        pad_x = message[2]&message[3]&message[4];
        BTSerial.println(pad_x);
        Serial.println(pad_x);
      }
    
    if(message.endsWith("d*\n")== true ){
    
        pad_x= BTSerial.parseInt();
        Serial.println(pad_x);
        BTSerial.println(pad_x);
    }
    
    while(message.endsWith("s*\n")== true){
        pad_y=-BTSerial.parseInt();
        Serial.println(pad_y);
        BTSerial.println(pad_y);  
      }
    }
}

Les buzzer actifs

Les buzzers électroniques sont alimentés en courant continu et équipés de circuits intégrés. Ils sont largement utilisés dans les ordinateurs, imprimantes, alarmes, jouets etc…. Il en existe deux types : les actifs et les passifs. Un buzzer actif possède un oscillateur intégré, il va donc générer un son lorsque le courant passe. Le principe de fonctionnement d’un buzzer passif est d’utiliser un signal PWM. Modulé à la bonne fréquence, la vibration génère différents sons. Exemple, envoyer une pulsation de 523Hz donne un Do, 587Hzdonne un Ré…

On différencie les deux type de buzzer grace au petit circuit imprimé visible sur le buzzer passif.
