Qu'est ce qu'arduino?

Arduino® est un ensemble matériel et logiciel qui permet d'apprendre l'électronique (en s'amusant) tout en se familiarisant avec la programmation informatique. Arduino est en source libre; vous pouvez donc télécharger le schéma d'origine et l'utiliser pour élaborer votre propre carte et la vendre sans payer des droits d'auteur.

Le materiel

Une carte Arduino est un cerveau qui permet de rendre intelligent des systèmes électroniques et d'animer des dispositifs mécaniques.

Les Arduino® sont des cartes électronique programable ( dotées d'un processeur et de mémoire) qui vont ressevoir des données émise par un capteur (temperature, humidité, potentiomètres...) et va transmettre des données "un message" à des recepteurs (leds, moteurs...). vous pouvez les trouver sous plusieur tailles et sous plusieurs formes, les principaux type de carte sont les arduino Nano, arduino Uno et les arduino Mega

Le logiciel

L'IDE Arduino est le logiciel qui permet de programmer les cartes Arduino. Les créateurs de Arduino ont développé un logiciel pour que la programmation des cartes arduino soit visuelle, simple et complète à la fois. C'est ce que l'on appelle une IDE, qui signifie Integrated Development Environment.

Le language de programation avec arduino

Un langage de programmation est un langage permettant à un être humain d'écrire un ensemble d'instructions (code source) qui seront directement converties en langage machine grâce à un compilateur (c'est la compilation). L'exécution d'un programme Arduino s'effectue de manière séquentielle, c'est-à-dire que les instructions sont exécutées les unes à la suite des autres.

Les structures de contrôle

Les structures de contrôle sont des instruction dans le language de programations. Elles sont utilisées pour commander le déroulement d'un programme el le devier dans une certaine direction

L'instruction Si (if)

Le terme Si fait appelle à une hypothèse ou à un cas de figure précis qui déterminerait une décision d'action. if exécutera son bloc d'instructions seulement si la condition définit est remplie.

if (condition 1; condition 2...){       //Si la conditon 1 et la conditon 2 sont validé 
    Instruction 1;                        //Alors le programme réalise l'instruction 1 et l'instruction 2 
    Instruction 2;
    ...
    }
  else if (condition 3; condition 4...){ //Sinon si la conditon 3 et la conditon 4 sont validé 
    Instruction 3;                       //Alors le programme réalise l'instruction 3 et l'instruction 4
    Instruction 4;
    ...
    }
  else{                                  //Sinon
    Instruction 5;                       //Le progrmme réalise l'instruction 5. 
    ...
    }

Les leds

Une LED est un composant capable d’émettre de la lumière lorsqu’il est parcouru par un courant électrique.

Elle ne laisse passer le courant électrique que dans un seul sens, pour que le courant puisse passer:

  • la cathode ("-" , patte courte) doit être connecté du coté de la masse (GND).
  • l’anode ("+", patte longue) doit être connecté du coté du port numérique.

Montage

Composants
  • Une carte Arduino
  • Une led rouge
  • Une resistance de 220Ω

Code

#define ledRouge 2            //Declaration de la led rouge sur la broche (pin) 2

void setup() {
  pinMode(ledRouge, OUTPUT);   //initialisation de ledRouge comme une sortie (output)
}

void loop() {
  digitalWrite(ledRouge, HIGH);//Allume la led, definie ledRouge sur le niveau haut
  delay(1000);                 //Le programme attend 1 seconde avant d'executer la prochaine instruction
  digitalWrite(ledRouge, LOW); //Etteind la led, definie ledRouge sur le niveau bas
  delay(1000);                 //Le programme attend 1 seconde avant d'executer la prochaine instruction
}
